## Objetivo do Projeto

O objetivo principal desta atividade foi aplicar os conceitos de estruturação de layouts modernos utilizando **HTML5 Semântico** e **CSS3**, simulando o ambiente visual de um e-commerce de grande porte. O foco do desenvolvimento evitou o uso excessivo de `<div>` genéricas, priorizando tags estruturais que favorecem SEO e tecnologias assistivas (leitores de tela).

## Tecnologias e Técnicas Utilizadas

* **HTML5:** Utilização de tags semânticas (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`) e atributos de acessibilidade (como `aria-label` e `aria-hidden`).
* **CSS3:** * Arquitetura modular (separação de responsabilidades com arquivos como `header.css`, `hero.css`, `products.css`, etc.).
  * Uso intensivo de **Flexbox** para alinhamentos unidimensionais (barras de navegação, ícones de benefícios).
  * Uso de **CSS Grid Layout** para a construção da vitrine de produtos de forma fluida e escalável.
  * Variáveis CSS (Custom Properties) para padronização de paleta de cores e espaçamentos.
  * Design Responsivo (Media Queries) para adaptação a dispositivos móveis.

## Uso de Inteligência Artificial

Para manter a transparência exigida no ambiente acadêmico, declaramos que ferramentas de Inteligência Artificial Generativa foram utilizadas durante o processo de desenvolvimento como um "pair programmer" assistente. A IA foi empregada para:

1. **Revisão e Refatoração:** Sugerir melhorias na separação de responsabilidades no CSS e apontar redundâncias no código.
2. **Mockup de Dados:** Geração de imagens placeholders e textos de preenchimento (lorem ipsum adaptado) para compor a vitrine de produtos sem quebrar o layout.

*Todas as decisões arquiteturais, estruturação de grid/flexbox e implementação final foram validadas e guiadas pelos autores do projeto.*

## Como executar

Este é um projeto estático puramente construído em HTML e CSS. Para visualizá-lo:

1. Faça o clone ou baixe o repositório.
2. Extraia os arquivos.
3. Abra o arquivo `index.html` diretamente em qualquer navegador moderno.
